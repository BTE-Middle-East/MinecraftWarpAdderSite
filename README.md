# Minecraft Warp Adder Site
An app that makes builders add warps to their builds manually without the help of staff members.

## What?
A basic form that takes coordinate and name input then writes it to the Minecraft server's ~/plugins/Essentials/warps/ directory.

## Why?
One word, laziness.
